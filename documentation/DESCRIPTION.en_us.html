Qalam Badi is a proportional text family for Bahá'í use in the widest possible sense, covering Latin, Arabic, Persian, Cyrillic and Greek, along with the Bahá'í Nine Pointed Star and the dot-below characters that transliterated Bahá'í names require.

It leans toward the hand of Mishkín-Qalam, the calligrapher and Apostle of Bahá'u'lláh — but it does so without stroke contrast. The family descends from Courier Prime by way of Courier Badi, and it keeps that ancestry's uniform monolinear stroke deliberately, so its calligraphic character has to come from proportion and rhythm instead: relative letter widths, the reach of a final's tail, how tightly letters join, how much air a word is given.

The spacing system is measured in the nuqta, the calligrapher's dot, which is the unit the classical Persian proportion system uses and which the source master already draws at a single shared size for both Arabic and Latin.

To contribute, see github.com/LaPingvino/qalam-badi.
